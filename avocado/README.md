# Avocado Dataset

## Description

Historical data on avocado prices and sales volume in multiple US markets.
Original dataset can be found here: [avocado prices](https://www.kaggle.com/neuromusic/avocado-prices/version/1#=)

### Technolegies

- Python
- Pandas
- Matplotlib
- Seaborn
